Wersja podstawowa:
* Możliwość dodawania/usuwania własnych przepisów, wyszukiwania przepisów, przechowywania
* Timer/Minutnik odliczający czas gotowania różnych wcześniej zdefiniowanych potraw


Co jest potrzebne do wykonania zadania:


* Formularz dla użytkownika:




Nazwa potrawy:
	type string
	Składniki:
	* Możliwość dodawania składnika 1 po 2, tak by tworzyły kompletną listę 
* Składnik musi był obiektem string + number (określa ilość składnika) + select (sztuk, gramów, kilogramów mililitrów, litrów)


	Sposób przygotowania:
	Pierwszy etap: type string
	Zdjęcie potrawy:
	Pierwszy etap: URL - type string
	Typ potrawy:
	5 x checkbox: Śniadanie, Obiad, Kolacja, Deser, Przekąska
	Czas przygotowania:
	type number
	

* Formularz będzie dodawać do pliku .json kolejne elementy listy, a także dodawać do struktury DOM odpowiednie elementy w postaci div
* Sortowanie przepisów po typie
* Sortowanie przepisów po składnikach z listy
* Wyszukiwarka przepisów:


Co masz w lodówce? [input]
	Select z opcjami typów potrawy
	Button [Pokaż]
	

* Główny widok ze wszystkimi przepisami: w widoku mobilnym 100% szerokość i 45vh (2 kafle z przepisami na 1 ekran). Kliknięcie w zdjęcie, nazwę lub button [Więcej] przenosi użytkownika do szczegółów danego przepisu.


* Kafel z przepisem:


Nazwa potrawy
	Zdjęcie potrawy
	Skrócony opis…..
Button [Więcej]
	

* Timer: pełnoekranowy widok zegara


Widoki:


1. Ekran powitania/loader: background + logotyp (z małą animacją migania?) na środku, zanim załaduje się zawartość strony
2. Strona główna: menu: [dodaj przepis, graj]; wyszukiwarka przepisów; główną część strony będą zawierać kafelki z najnowszymi przepisami
3. Strona pojedynczego przepisu: nazwa, składniki, czas przygotowania, zdjęcie, treść przepisu, button [gotuj] -> przekierowanie do timera
4. Timer: pełnoekranowy widok zegara odliczającego czas do zakończenia gotowania


Dodatkowo:
* Gra memory na zabicie czasu oczekiwania na potrawę
